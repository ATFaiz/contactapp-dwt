# Short Description of ConactList App

Contact List App is a web application built using React.js.
It allows users to create, view, edit, and delete contacts.
The app utilizes IndexedDB for data storage, providing offline access and data persistence.
It incorporates features such as camera integration for adding contact profile pictures.
Geolocation API is used to display the location of contacts on a map.
The app utilizes Mapbox API for map rendering and geocoding.
It implements client-side routing using React Router for seamless navigation.
The UI is designed to be responsive and user-friendly, with Bootstrap as the primary CSS framework.
The app leverages service workers to enable offline access and cache management.
It showcases the use of modern front-end technologies and best practices in web development.
